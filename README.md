# FSAD-HACKATHON: Online Voting Management System

A secure, digital voting platform designed for transparency and ease of use. This project features a full-stack architecture with a dedicated voter authentication flow and real-time candidate selection.

## 🚀 Project Overview
**Secure Vote** is a web-based application that allows users to cast their votes securely after a multi-factor authentication process.

### Key Features:
* **Voter ID Verification:** Initial validation of voter credentials.
* **OTP Authentication:** Secure one-time password flow to prevent unauthorized access.
* **Dynamic Candidate Directory:** Interactive dashboard to view and select candidates.
* **Modular Codebase:** Separated frontend and backend folders for better scalability.

---

## 📸 User Interface Walkthrough

### 1. Voter Login
Users start by entering their unique Voter ID to request an authentication code.

### 2. OTP Verification
A secure 2-step verification process to ensure the identity of the voter.

### 3. Voting Dashboard
The main interface where voters can browse candidates by party and cast their vote with a single click.

---

## 🛠️ Repository Structure
The project is split into two main directories as shown in the repository root:

* `voting-backend`: Contains the server-side logic, API endpoints, and database configurations.
* `voting-frontend`: Contains the React/Vite source code and UI components.

---

## ⚙️ How to Run

### Backend
1. Navigate to `voting-backend`
2. Run `npm install`
3. Start the server using `npm start` (Runs on port 8080 by default)

### Frontend
1. Navigate to `voting-frontend`
2. Run `npm install`
3. Launch the dev server: `npm run dev`
4. Open your browser at `http://localhost:5173`
